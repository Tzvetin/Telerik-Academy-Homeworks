# Fluent Interface

Този шаблон представлява добавка към вече съществуващ код, която го прави по-лесен за използване. С използването му се намалява синтактичния шум в кода – непрекъснатото прехвърляне на стойност между променливи. Това се реализира чрез прехвърляне на текущото състояние на даден обект през една верига (chaining). Често за целта се ползва символа “.”, който свързва каскадно последователните етапи. По този начин и самият код става по-четим и разбираем. Fluent Interface се използва все по-често за създаване на нови обекти като предоставя един по-интуитивен подход при инициализацията им.
